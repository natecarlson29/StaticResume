Oldschool Runescape Skill Calculator

PHP / MySQL / HTML / CSS

This is a website that I wrote for the game Old School Runescape. It connects to the game's API to retrieve stats on the players accounts and then will display to you different ways to train your skills based on the skill you chose and what level you have in that skill.
